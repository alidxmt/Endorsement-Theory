Endorsement Theory

Ali Zolfagharian - 'What If' Research Group - Konstanz University - 2019

Endorsement theory, ET, is a formal epistemological theory which represents an epistemic state of an agent by an acceptance base function (ABF), an endorsement critical point (ECP), and an endorsement function (EF). ABF, ECP, and EF respectively depend on agent’s evidence, context and reasons.

The static ET ends by representing qualitative and quantitative doxastic attitudes, namely; belief, disbelief, suspense and ignorance. For the dynamic part, I explain how an agent can update her acceptance base function, endorsement threshold and her way of constructing the endorsement function.

The last part is about conditionals and reason. Conditional degree of endorsement of proposition, depends on the given condition, a condition could be change in the acceptance base. Reason on the other side is about comparing how a condition can change the degree of endorsement of the proposition and its complement.

Preliminary definitions and notation
Set of possibilities and propositions

Let W be the set of all possibilities and n be the number of all possibilities. Then Pi ⊆ W is a proposition, a set of possibilities; such that the subscript i is a decimal number which its binary form indicates which possibilities are in Pi. The notation, using subscript i, improves the readability of the text and it is easy to use this notation for a programming language like bitwise operator. Also, it reflects the definition of a proposition, a function from the set of possibilities to {0,1}.

  • (1) W = { w 1 , w2 ,…, wn }
  • (2) Pi ⊆ W

Let me explain with an example: Assume W={ w1 , w2 , w3 , w4 }, then the proposition W is P15 which is P(1111), the number 1111 tells that all possibilities w1,, w2,, w3,, w4 belongs to P15. Similarly, P5 ≡ P(0101) which tells that P5 contains w1,, w3. And P0 means the empty set.

Frames and ground

The set of frames F is the set of all algebra over W. Fi is a subset of F, that all its member contains Pi. In other word Fi is the set of all algebra over W which have Pi as its member. An algebra could be thought as an inner context or a framework that has enough position to receive evidences. There might be cases that the inner context needs to be updated.

  • (3) F = { Bi | Bi is an algebra and a subset of 2W }
